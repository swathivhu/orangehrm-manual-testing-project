# OrangeHRM Manual Testing Project

## Overview

This project demonstrates end-to-end Manual Testing of the OrangeHRM application. The project covers requirement analysis, test scenario design, test case creation, test execution, and defect reporting.

## Application Under Test

OrangeHRM is a Human Resource Management System used to manage employee information, leave requests, recruitment processes, and user administration.

## Modules Tested

* Login
* Dashboard
* Admin
* PIM (Personnel Information Management)
* Leave
* Recruitment
* My Info
* Logout

## Testing Types Performed

* Functional Testing
* Validation Testing
* Negative Testing
* UI Testing
* Security Testing
* Session Management Testing
* Compatibility Testing
